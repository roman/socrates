  start, before implementation, before commit (code-critic foreground/opus,
  max 2 rounds), before ending session (docs first, then commit everything).
- **6.6 .msgs/ inbox**: Async human→agent communication with archive workflow.
- **6.7 .ralph-stop**: Graceful loop exit.

Also fixed `/init` command to copy RALPH.md and discipline gates from template
files instead of generating inline content.

## Key Decisions

1. RALPH.md is a template installed by `/init` — projects can customize.
2. Discipline gates specify code-critic as foreground agent with opus model
   per user preference for thorough review.
3. End-of-session gate: docs/ADRs first, then commit everything together.
4. `/init` now copies from templates rather than generating inline, keeping
   a single source of truth.
