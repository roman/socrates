## Learnings

- choo-choo-ralph uses `bd ready --json` for machine-readable output; tk's
  `tk ready` outputs plain text (ID as first field, parseable with awk)
- tk has no `--json` flag on `ready` — use `tk query` with jq for structured data
- devenv `files` attribute uses inline `text`, not symlinks — must `builtins.readFile`
  from the nix store package, not `source` symlink
- nixDir auto-discovers packages but skills map needs a dummy derivation with
  `passthru` (same pattern as zoo.nix's `claude-skills`)
- nixfmt pre-commit hook reformats on first commit attempt — just re-stage and retry
