## Learnings

- `tk query` outputs NDJSON (one JSON object per line), not a JSON array.
  Use `jq -s` to slurp into array for filtering, or process line-by-line.
- Claude Code hooks provide absolute file paths in `tool_input.file_path`,
  not relative. Grep substring matching handles this but exact-match logic
  would need path normalization.
- `set -e` in test scripts conflicts with grep-based assertions (grep
  returns exit 1 on no match). Use `set -uo pipefail` without `-e` for
  assertion scripts.
- `git log --format='%B'` splits poorly for multi-paragraph commits.
  Iterate by SHA with `--format='%H'` instead.
